1. `display: none`

通过 CSS 操控 display，移出文档流

```css
display: none;
```

2. `opacity: 0`，透明度为 0，仍在文档流中，当作用于其上的事件(如点击)仍有效

```css
opacity: 0;
```

3. `visibility: hidden`
   透明度为 0，仍在文档流中，但作用于其上的事件(如点击)无效，这也是 visibility:hidden 与 opacity: 0 的区别

```css
visibility: hidden;
```

4. `content-visibility`
   移出文档流，但是再次显示时消耗性能低

```css
content-visibility: hidden;
```

5. 绝对定位于当前页面的不可见位置

```css
position: absolute;
top: -9000px;
left: -9000px;
```

6. 字体大小设置为 0

```css
font-size: 0;
```
